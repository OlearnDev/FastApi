---------------------- Creer env
pip install virtualenv  : installer virtualenv
virtualenv venv  		: Crée un nouvel env nommé venv
pip install api

.\venv\Scripts\activate	: Active l'env
pip install fastapi
pip install uvicorn		: Serveur pour executer api et afficher sur une page web 
uvicorn api:app --reload : Fournit un serveur pour lancer le code dans api.py 
                            et fournit l'adresse web pour voir le resulat d'execution
